# Devoir_Portfolio_ReactJS_Ghomari

Devoir Portfolio avec React-JS

Portfolio de John Doe:

Ce projet est un portfolio développé avec React et Bootstrap.
Il présente le profil, les compétences et les projets de John Doe.

Objectif:

Créer un site vitrine moderne et responsive pour présenter :

- Une présentation personnelle
- Les compétences
- Les projets réalisés
- Un formulaire de contact

Technologies utilisées:

- React
- Bootstrap
- JavaScript
- HTML / CSS

---

:Prérequis

Avant d’installer le projet, vous devez avoir :

- Node.js (version 18 ou supérieure recommandée)
- npm (installé automatiquement avec Node.js)

Pour vérifier :

```bash
node -v
```
